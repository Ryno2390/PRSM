**Fix 2 — IPFS daemon auto-start**
- File: `prsm/node/node.py`
- Added `_ensure_ipfs_available()` async method that:
  - Checks if daemon is already running via HTTP POST to `http://127.0.0.1:5001/api/v0/id`
  - If not running and `ipfs` binary is on PATH, starts daemon with `subprocess.Popen(["ipfs", "daemon", "--init"])`
  - Waits up to 10 seconds for daemon to become ready (polls every 0.5s)
  - Logs clear messages for all cases; never blocks node startup on failure
- Wired into `start()` method; cleanup in `stop()` terminates daemon if we launched it
