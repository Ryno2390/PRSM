### Additional observations

- **Burn linearity:** 0.43 μETH/day across 7 daily cron runs with **zero variance** (sub-rounding-error noise only). This is a strong signal that the synthetic workload is deterministic and the Sepolia RPC path is stable. Extrapolated to mainnet at comparable gas prices, 1 μETH/day ≈ $0.003/day at current ETH prices for the synthetic workload — negligible.
- **No intervention required:** zero manual interventions, zero alert conditions, zero restarts of the cron job across the window. The ops profile is as boring as expected and planned for.
- **Contract verification holding:** both ProvenanceRegistry (`0x3744D1104c236f0Bd68473E35927587EB919198B`) and RoyaltyDistributor (`0x95F59fA1EDe8958407f7b003d2B089730109BD54`) have held their Basescan verification status across the full window. No source-code drift, no re-verification events.
- **Day 0 smoke test held:** the `register_content` transaction from Day 0 (`0x9a43de4e...`) remains queryable; the L2 state propagation observed at ~2 seconds on Day 0 has not regressed.
