Files NOT vendored
------------------
The following upstream directories were intentionally not vendored:

    src/parallax/             — runtime; PRSM uses TensorParallelExecutor
    src/backend/              — GPU runtime; out of scope
    src/frontend/             — chat UI; PRSM has its own SDK
    src/parallax_extensions/  — out of scope
    src/parallax_utils/       — only minimal helpers ported into
                                _vendored_utils.py; full module not vendored
    src/router/               — out of scope (not consumed by scheduling/)

The upstream `lattica` Python package, which Parallax declares as a
runtime dependency, is NOT pulled into PRSM. The new (τ, ρ) profile
DHT introduced by Phase 3.x.6 Task 4 runs over PRSM's existing Phase 6
transport (TCP / SOCKS adapters with EIP-4361 wallet binding).
